J'espère que tu vas bien !

Pour que le formulaire de contact de ton site fonctionne et envoie les messages directement sur ta boîte mail, j'ai besoin que tu te crées un compte rapide sur un outil gratuit (Formspree).

Ça prend littéralement 2 minutes chrono, voici les étapes :

1. Inscription

Va sur formspree.io, clique sur Sign Up et inscris-toi avec l'adresse e-mail où tu veux recevoir les messages de tes clients.

2. Créer le formulaire

Une fois sur ton tableau de bord :

    Clique sur le bouton + Add New en haut à gauche.

    Choisis New Form dans le petit menu.

3. Configurer et valider

Une petite fenêtre s'ouvre :

    Dans Form Name, mets ce que tu veux.

    Vérifie que c'est bien ton adresse mail qui est écrite dans Send emails to.

    Clique sur le gros bouton rouge Create Form.

4. Le fameux code

Dernière étape ! Tu vas tomber sur une page avec une ligne FORM ENDPOINT :

    Clique sur le bouton Copy juste à côté du lien (qui ressemble à https://formspree.io/f/mgobzwyz )

    Colle-moi ce lien en réponse à ce message.

Dès que j'ai ça, je t'intègre ça sur le site et on fait un test !
